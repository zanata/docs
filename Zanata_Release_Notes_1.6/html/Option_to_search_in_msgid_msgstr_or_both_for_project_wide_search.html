2.3.9. Project-wide search: Search in msgid, msgstr, or both

Searches in Zanata's web editor would search all strings, both source and target strings.
The new search dialog gives users the ability to specify what content they wish to search: source strings only, target strings only, or both.
For more information see https://bugzilla.redhat.com/show_bug.cgi?id=817453.

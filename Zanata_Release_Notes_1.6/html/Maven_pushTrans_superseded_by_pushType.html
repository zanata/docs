2.2.3. Maven pushTrans superseded by pushType

The maven -Dzanata.pushTrans option is now deprecated and is replaced by -Dzanata.pushType, which can take the following values:
source
Push source documents only (This is the default value when not specified)
trans
Push translations only
both
Push both source documents and translations
If the now deprectated pushTrans option is used, it is interpreted as -Dzanata.pushType=both.
More Information
For more information see https://bugzilla.redhat.com/show_bug.cgi?id=736898.
